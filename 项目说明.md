# 蓝牙控制器项目说明

## 当前目录包含两个版本

### 1. 微信小程序版本
- `pages/` - 微信小程序页面
- `app.js`、`app.json`、`app.wxss` - 小程序入口文件

### 2. React Native Android APP 版本（最新更新）
- `App.js` - React Native 主程序（已优化按钮布局）
- `android/` - Android 原生配置
- `package.json` - 依赖配置

---

## 按钮名称已更新

React Native 版本的按钮布局已优化：

| 布局位置 | 按钮显示 | 发送字符 | 颜色 |
|---------|---------|---------|------|
| 顶部居中 | 上 (w) | `w` | 蓝色 |
| 左侧 | 左 (a) | `a` | 绿色 |
| 右侧 | 右 (d) | `d` | 红色 |
| 底部居中 | 下 (s) | `s` | 橙色 |
| 功能区左 | 返回 (q) | `q` | 紫色 |
| 功能区右 | 确认 (f) | `f` | 青色 |

---

## 使用 React Native 版本

### 快速运行

```bash
# 安装依赖
npm install

# 连接 Android 设备后运行
npm run android
```
